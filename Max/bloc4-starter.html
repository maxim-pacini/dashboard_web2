CH 01 Les sélecteurs

+

jQuery permet de cibler des éléments du DOM avec la même syntaxe que CSS. Tout passe par $().

// Sélection par classe $(".item").css("background", "#ffd23f");
Résultat — tous les .item
  • Item A
  • Item B
  • Item C
// :eq(index) — un seul élément $(".item2").eq(1).addClass("highlight");
Résultat — seulement le 2e
  • Premier
  • Deuxième
  • Troisième
À retenir : $(this) dans un événement représente l'élément qui a déclenché l'action.

CH 02 Les événements

+

Click, hover, keypress… jQuery unifie tous les événements avec .on() et ses raccourcis.

// .click() $("#btn").click(function() { $(this).text("Cliqué !"); });
Clique sur le bouton
// .hover(entrer, sortir) $("#box").hover( function() { /* survol → vert */ }, function() { /* sortie → rouge */ } );
Survole le carré
// .on("keyup") — frappe clavier $("#input").on("keyup", function() { $("#output").text($(this).val()); });

CH 03 Effets & animations

+

jQuery propose des animations prêtes à l'emploi : fadeIn, slideUp, animate, toggleClass…

// .fadeToggle(600) $("#box").fadeToggle(600);
Disparaît / réapparaît
// .slideToggle(500) $("#box").slideToggle(500);
Se déplie / replie
// .animate() custom $("#box").animate({ "margin-left": "180px", "opacity": 0.4 }, 1000);
Se déplace vers la droite
Astuce : Les durées sont en ms. 400 = par défaut, "slow" = 600ms, "fast" = 200ms.
